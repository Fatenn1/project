# Project Images

Add your project images here with the following naming convention:

- `club-securinets.jpg` - Club Securinets project image
- `3afssa.jpg` - 3afssa.tn project image
- `luxury-brand.jpg` - Luxury Brand Identity project image
- `ecommerce.jpg` - E-Commerce Platform project image
- `mobile-app.jpg` - Mobile App Design project image
- `editorial.jpg` - Editorial Design project image

## Image Recommendations:
- Recommended size: 1200x900px (4:3 aspect ratio)
- Format: JPG or PNG
- File size: Optimized for web (under 500KB recommended)
- Quality: High resolution for best display

The portfolio will automatically display these images when added to this folder.
