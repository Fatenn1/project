# Customer Photos

Add customer testimonial photos here with the following naming convention:

- `sarah-johnson.jpg` - Sarah Johnson (Creative Director at Luxury Brands Co.)
- `michael-chen.jpg` - Michael Chen (Product Manager at Tech Innovations)
- `emma-williams.jpg` - Emma Williams (Marketing Lead at Startup Hub)

## Image Recommendations:
- Recommended size: 200x200px (square, 1:1 aspect ratio)
- Format: JPG or PNG
- File size: Optimized for web (under 100KB recommended)
- Quality: High resolution for best display
- Style: Professional headshot or portrait photos work best

The testimonials will automatically display these photos when added to this folder. If a photo is missing, the system will fallback to the emoji avatar.
